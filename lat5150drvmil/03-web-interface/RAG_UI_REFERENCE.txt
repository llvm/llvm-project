INTERACTIVE ELEMENTS
═══════════════════════════════════════════════════════════════════════════════

HEADER CLICK:
  • Toggle panel expand/collapse
  • Indicator changes: [+] ↔ [-]

INPUT FIELDS:
  • Click to focus
  • Type file/folder path or search query
  • Press Enter to submit
  • Auto-clear on success (file/folder inputs)

BUTTONS:
  • FILE:    Add single file to RAG
  • FOLDER:  Add all files in folder to RAG
  • FIND:    Search RAG database
  • REFRESH: Reload stats and document list

WORKFLOW EXAMPLES
═══════════════════════════════════════════════════════════════════════════════

DAILY RESEARCH WORKFLOW:
1. Open terminal interface
2. Panel shows existing docs (e.g., "apt_tradecraft.txt - 5,264T")
3. Download new paper to /home/john/research/new_paper.pdf
4. Type path in file input: /home/john/research/new_paper.pdf
5. Press Enter
6. See "File added successfully: 2,341 tokens"
7. Stats update: DOCUMENTS: 4, TOKENS: 7,605
8. Paper now searchable

TECHNIQUE LOOKUP:
1. Type in search: "persistence mechanisms"
2. Press Enter
3. See ranked results from all indexed papers
4. Read content previews to find what you need
5. Reference original files if you need full context

BATCH IMPORT:
1. Organize papers in /home/john/research/apt_papers/
2. Type folder path in folder input
3. Press Enter
4. Wait for processing (may take 30s-2min for large folders)
5. See detailed summary: "Folder processed: 15 files, 23,412 tokens"
6. View list of added files
7. Stats show all new content
